üìä Key Performance Indicators

Pipeline Details Table

Navigate and use the detailed pipeline view

üìã Pipeline Details

The Pipeline Details table shows all candidates in your pipeline with key information at a glance. This is where you'll spend most of your time managing candidates.

üì∏
Screenshot: Pipeline Details Table

Table Columns

Column Description
Candidate Name Clickable link to candidate record
Status Current pipeline stage (New, In Process, Hired, etc.)
Highest Level Furthest interview stage reached
Next Interview Date/time of next scheduled interview
Days in Stage How long in current status
Source How candidate was sourced
Actions Quick action buttons

üîç Filtering & Sorting

Use the table controls to find specific candidates quickly:

  • Search Box: Type to filter by candidate name
  • Column Headers: Click to sort ascending/descending
  • Status Filter: Filter by pipeline stage
  • Date Range: Filter by date added or interview date
üí° Quick Actions Hover over a row to see quick action buttons: Schedule Interview, Update Status, and View Details.
