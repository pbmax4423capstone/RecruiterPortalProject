üì¶ Install the Extension

Add the LinkedIn Import extension to your browser

üìç Installation Steps (Edge)

  1. Open Microsoft Edge
  2. Go to edge://extensions
  3. Enable Developer mode (toggle in bottom left)
  4. Click "Load unpacked"
  5. Select the edge-extension-linkedin folder
  6. The extension icon appears in your toolbar!

üì∏ Screenshot: Edge extensions page with Developer mode enabled

üìç Installation Steps (Chrome)

  1. Open Google Chrome
  2. Go to chrome://extensions
  3. Enable Developer mode (toggle in top right)
  4. Click "Load unpacked"
  5. Select the chrome-extension-linkedin folder
  6. The extension icon appears in your toolbar!

Pin the Extension Click the puzzle piece icon in your toolbar and pin the LinkedIn Import extension for easy access.

Connect the extension to your Salesforce org

üìç First-Time Setup

  1. Click the extension icon in your toolbar
  2. Click "Settings" or the gear icon
  3. Enter your Salesforce Domain (e.g., massmutual.my.salesforce.com)
  4. Click "Connect to Salesforce"
  5. Log in when prompted
  6. Authorize the connection

üìç Action Steps

üöÄ Import to Salesforce

Create the candidate with one click

üìç Import Steps

  1. Click the extension icon in your toolbar
  2. Review the captured information
  3. Select Lead Source = "LinkedIn"
  4. Click "Import to Salesforce"
  5. Wait for the success confirmation!
