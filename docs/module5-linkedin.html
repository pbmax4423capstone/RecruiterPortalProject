Extension Overview

Introduction to the LinkedIn import extension

üîó What is the LinkedIn Extension?

The LinkedIn Import Extension is a browser add-on that allows you to quickly import candidate information directly from LinkedIn profiles into the Recruiter Portal. It creates both a Contact record and a linked Candidate record in Salesforce.

üîò Save to Salesforce Button When you visit a LinkedIn profile, a blue "Save to Salesforce" button appears below the person's profile photo. Click this button to open the extension popup and capture their information.

Using the Extension

How to import candidates from LinkedIn

üì§ Importing a Candidate

Follow these steps to import a candidate from LinkedIn:

Navigate to the candidate's LinkedIn profile page
Click the blue "Save to Salesforce" button below their profile photo
Review the captured data in the popup window - verify name, email, phone, and birthday are correct
Make any necessary edits or additions to the fields
Click the "Create Candidate" button
A Contact and linked Candidate record will be created in Salesforce
üí° How It Works When you click the "Save to Salesforce" button, the extension automatically opens a popup displaying the candidate's profile information. From this popup, you can review the data and save the candidate directly to Salesforce.

‚úèÔ∏è Editing Before Import

The extension popup allows you to review and modify data before importing:

  • First Name / Last Name - Verify the name split is correct (designations like CFP¬Æ, MBA are automatically removed)
  • Email - Captured automatically if visible in Contact Info modal
  • Phone - Captured automatically from Contact Info (800 numbers are excluded)
  • Birthday - Captured automatically if the person has added it to their profile
üí° Success Message After a successful import, you'll see a green confirmation message. The Contact and Candidate records are now created in Salesforce.
‚ö†Ô∏è Duplicate Detection If a candidate with the same name already exists, you'll see an error message. Search for the existing record in Salesforce instead of creating a duplicate.

Data Mapping

Understanding how LinkedIn data maps to Salesforce fields

üîÑ Field Mapping

The extension creates a Contact record first, then a linked Candidate record. Here's how data is mapped:

Contact Record Fields

LinkedIn Field Contact Field Notes
Full Name First Name, Last Name Split automatically, designations removed
Email (Contact Info) Email From mailto: link in Contact Info modal
Phone (Contact Info) Phone From Contact Info modal (800 numbers excluded)
Birthday (Contact Info) Birthday If available in their profile
Profile URL Website LinkedIn profile URL stored here

Candidate Record Fields

Field Value Notes
Contact Linked Contact Links to the Contact record created above
First Name, Last Name From LinkedIn Same as Contact
Personal Email Formula Field Automatically pulls from Contact.Email
Mobile From Contact Displays Contact's Phone on Candidate page
Website LinkedIn URL Profile URL for reference
Status 0-Prequal Auto-set for new candidates

üìã Single Source of Truth

The Contact record is the single source of truth for personal information:

  • Email - Stored on Contact, displayed on Candidate via formula field
  • Phone - Stored on Contact, displayed on Candidate record page
  • Birthday - Stored on Contact record

If you need to update email or phone, update the Contact record and it will automatically reflect on the Candidate.

Troubleshooting

Common issues and how to resolve them

üîß Common Issues

"Save to Salesforce" Button Not Appearing

  • Verify you're on a LinkedIn profile page (not search results)
  • Wait a few seconds for the page to fully load
  • Try refreshing the LinkedIn page
  • Check that the extension is enabled in your browser

"Session Expired" or Authentication Error

  • Your Salesforce session has expired
  • Open Salesforce in a new tab and log in
  • Return to LinkedIn and try again
  • If still not working, click "Settings" in the extension and re-connect

Email/Phone Not Captured

  • Click "Contact info" on the LinkedIn profile to open the modal
  • Wait for the Contact Info modal to fully load
  • The person may not have added email/phone to their profile
  • Some users have privacy settings that hide this information

"Duplicate" Error Message

  • A Contact or Candidate with the same name already exists in Salesforce
  • Search for the existing record in Salesforce
  • Do not create duplicates - use the existing record

Name Includes Credentials (CFP¬Æ, MBA, etc.)

  • The extension automatically removes common designations
  • If credentials still appear, manually edit the name in the popup before saving

üìû Getting Help

If you continue to experience issues:

  1. Take a screenshot of any error messages
  2. Note which browser and version you're using
  3. Contact IT Help Desk with the details
  4. Reference "LinkedIn Import Extension" in your ticket
üí° Keep Extension Updated IT will periodically release updates to the extension. When notified, re-install using the latest version for best performance.
