üìÖ Handle Contract Extensions

Process extensions for agents who need more time

üìã Extension Eligibility

  • Agent has shown consistent effort
  • Progress is close but not quite at requirements
  • Maximum extension: 4 months
  • Requires management approval

üìç Grant an Extension

  1. Open the agent's Candidate record
  2. Click Edit
  3. Set Extension Granted = Yes
  4. Enter Extension Granted Date
  5. The Extended End Date auto-calculates
  6. Click Save
‚ö†Ô∏è
Important! Once an extension is granted, the Days Remaining counter resets based on the new Extended End Date. Document the reason for extension in Notes.

üéØ Manage Contract Outcomes

Process conversions to Contract A or terminations

‚úÖ Conversion to Contract A

Converted to A

  1. Verify agent has met both requirements ($2,500 FYC + 5 submissions)
  2. Open the Candidate record
  3. Set Contract Outcome = "Converted to Contract A"
  4. Set Transition to A Date
  5. Update Contract Type to "Contract A"
  6. Save the record
üéâ
Congratulations! When an agent converts to Contract A, they become a permanent financial advisor. This is the goal!

‚ùå Termination

Terminated

  1. Open the Candidate record
  2. Set Contract Outcome = "Terminated"
  3. Set Termination Date
  4. Select Termination Reason
  5. Update Status to "Terminated"
  6. Save the record
‚ö†Ô∏è
Termination Reasons
  • Did Not Meet Requirements
  • Voluntary Resignation
  • Performance Issues
  • Other (specify in Notes)
