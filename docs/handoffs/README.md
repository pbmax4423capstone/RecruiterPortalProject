# Handoff Documents

This directory contains handoff documentation when work is passed between Patrick, Cole, or their agents.

## Purpose

When completing a phase of work that needs to be handed to another person or agent, create a handoff document here using the template from [WORK_COORDINATION.md](../../WORK_COORDINATION.md).

## Naming Convention

`HANDOFF-[TaskID]-[Date]-[From]-to-[To].md`

Examples:

- `HANDOFF-042-2026-01-08-Cole-to-Patrick.md`
- `HANDOFF-043-2026-01-09-PatrickAgent-to-ColeAgent.md`

## Template

See [WORK_COORDINATION.md](../../WORK_COORDINATION.md) for the full handoff template.
